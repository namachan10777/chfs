# NAME
chlist - list CHFS servers

# SYNOPSIS
**chlist** [_server_]

# DESCRIPTION
**chlist** lists CHFS servers.

# ENVIRONMENT
CHFS_SERVER
: one of server addresses

CHFS_RPC_TIMEOUT_MSEC
: RPC timeout in milliseconds

CHFS_LOG_PRIORITY:
: max log priority to report
