# NAME
chmkdir - create a directory in CHFS

# SYNOPSIS
**chmkdir** _mode_ _directory_

# DESCRIPTION
**chmkdir** creates a directory in CHFS.

# ENVIRONMENT
CHFS_SERVER
: one of server addresses

CHFS_RPC_TIMEOUT_MSEC
: RPC timeout in milliseconds

CHFS_LOG_PRIORITY:
: maximum log priority to report
