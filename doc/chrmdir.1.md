# NAME
chrmdir - remove a directory in CHFS

# SYNOPSIS
**chrmdir** _directory_

# DESCRIPTION
**chrmdir** removes a directory in CHFS.

# ENVIRONMENT
CHFS_SERVER
: one of server addresses

CHFS_RPC_TIMEOUT_MSEC
: RPC timeout in milliseconds

CHFS_LOG_PRIORITY:
: maximum log priority to report
